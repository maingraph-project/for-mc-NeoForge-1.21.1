# 0.3.6：蓝图分享更新 (Blueprint Sharig Update)

- **新增**：
    - **导出到 .mgmcbp**：现在可以右键点击蓝图列表项，将其导出为 `.mgmcbp` 文件。导出的文件内置了该蓝图的所有 ID 映射（Mapping）设定与逻辑数据。
    - **导入 .mgmcbp**：新增导入功能，支持从 `.mgmcbp` 文件导入蓝图。
- **其他**：
    - 导出的文件将自动保存至 `mgmc_blueprints/exports/` 目录。
    - 导入前只需将文件放入 `mgmc_blueprints/imports/` 目录即可在 UI 中选择并导入。

# 0.3.6: Blueprint Sharing Update

- **New**:
    - **Export to .mgmcbp**: You can now right-click on a blueprint list item to export it as a `.mgmcbp` file. The exported file includes all ID mapping settings and logic data for that blueprint.
    - **Import .mgmcbp**: Added import functionality, supporting blueprint imports from `.mgmcbp` files.
- **Others**:
    - Exported files are automatically saved to the `mgmc_blueprints/exports/` directory.
    - Simply place files in the `mgmc_blueprints/imports/` directory to select and import them via the UI.
