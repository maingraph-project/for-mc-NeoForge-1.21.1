## v0.3.3：“排列与无界”更新 (Layout & Boundless Update)

- **优化**:
    - **重构自动布局算法**: 引入 Sugiyama 算法实现动态间距与智能排序，解决节点重叠问题并大幅减少连线交叉。
    - **交互安全增强**: 为顶部栏“自动布局”与“重置视角”添加 1s 长按确认逻辑，并实时显示进度反馈，防止误触。
    - **视觉动画提升**:
        - 为“重置视角”添加平滑过渡动画，使视角切换更加自然。
        - 为“自动布局”添加节点位移插值动画，实时演示节点从杂乱到有序的排列过程。
    - **连线交互**: 支持从端口拖出连线并在空白处释放时自动弹出节点列表。
    - **输入限制解除**: 将节点配置输入、标记节点文本的最大字符限制大幅提升（最高支持 32,767 字符）。

## v0.3.3: "Layout & Boundless" Update

- **Optimizations**:
    - **Refactored Auto-Layout Algorithm**: Introduced the Sugiyama algorithm to achieve dynamic spacing and intelligent sorting, resolving node overlapping and significantly reducing edge crossings.
    - **Enhanced Interaction Safety**: Added a 1s long-press confirmation logic for "Auto-Layout" and "Reset View" in the top bar, with real-time progress feedback to prevent accidental triggers.
    - **Visual Animation Improvements**:
        - Added smooth transition animations for "Reset View" for a more natural perspective switch.
        - Added node displacement interpolation animations for "Auto-Layout" to demonstrate the transition from a cluttered to an organized arrangement in real-time.
    - **Connection Interaction**: Support for dragging out a connection from a port and automatically popping up the node list when released in an empty space.
    - **Input Limit Removal**: Significantly increased the maximum character limit for node configuration input and marker node text (up to 32,767 characters).
